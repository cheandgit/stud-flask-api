# Практическое задание: Деплой Flask API (простой Flask + JSON API)

## Цель: Развернуть работающее веб-приложение в интернете

## Структура проекта stud-api-project
- app.py              (Основной файл)
- requirements.txt    (Зависимости)
- README.md

## Задачи:
1. Создайте репозиторий на GitHub
2. Откройте его в Codespaces
3. Создайте файлы app.py, requirements.txt и README.md
4. Запустите приложение локально в Codespace
5. Задеплойте на Render
6. Пришлите преподавателю:
   - Ссылку на GitHub репозиторий
   - Ссылку на работающее приложение на Render
   - Скриншот работающего API (проверка всех эндпоинтов)

## Критерии оценки:
✅ Приложение работает локально в Codespace
✅ Приложение доступно по публичной ссылке Render
✅ Все эндпоинты API работают
✅ Код выложен на GitHub
